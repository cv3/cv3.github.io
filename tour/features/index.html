Features

CV3 is packed with features that modern online retailers demand. Take a look at this bulleted list of features to get an idea of the depth of functionality that could be enabling your online business to thrive and set you apart from the competition.

Platform Specs & Certifications

  • fast & flexible cloud-based application
  • multi-server, multi-datacenter
  • dedicated SQL-based cloud storage
  • edge CDN for binary assets
  • global, multi-homed DNS
  • on-demand resource scaling
  • PCI-DSS Level 1 VISA Certification
  • full backup rotation & code escrow

Customization & Branding

  • 100% editable site, including checkout
  • raw HTML & CSS & javascript access
  • open source theming engine (Smarty)
  • HTML5 & CSS3 enabled by default
  • Responsive mobile design
  • HTML & JS framework friendly
  • Legacy browser support via Modernizr
  • PHP-style code support
  • re-usable “building blocks” of code
  • staging area for reviewing changes
  • internal file and image libraries
  • A/B testing, including Google W.O.

Web Shopping

  • flexible product search
  • search filtering and results ordering
  • optional support for SLI Advance Search
  • nested category browsing
  • multiple-product images & videos
  • support for image zoom & effects
  • product ratings & reviews
  • optional inventory control
  • related products & required products
  • parent / child products
  • product attributes & sets
  • volume pricing discounts
  • minimum quantity limits
  • customizable products
  • product assemblies

Mobile Shopping

  • Responsive, mobile-first design
  • use HTML5, JQuery Mobile and more…
  • target multiple mobile platforms
  • shopper toggle between mobile & web
  • configurable default for tablets
  • ajax-based content loading
  • full shopping & expedited checkout
  • theme updates retain customizations

Social Shopping

  • Facebook like buttons
  • product ratings & reviews
  • share buttons, e-mail products to friends
  • Embedded shopping in Facebook

International Features

  • language-specific theme support
  • set language-specific currency
  • international shipping
  • international payments

Traffic-building / SEO

  • customizable, friendly URLs
  • canonical URLs
  • customizable meta-data per page
  • additional user-defined page data
  • redirect mapping
  • landing pages with add-to-cart
  • Google Site Map support

Membership

  • view previous orders / track orders
  • quick-reorder / address book
  • view gift certificate balance

Marketing & Promotions

  • catalog request & e-mail sign-up
  • referrer tracking & media code support
  • add-to-cart dynamic landing pages
  • segmented customer groups
  • behavioral group triggers
  • individual & group promo codes
  • single-use promo codes
  • fixed & percentage promos
  • category-based promos
  • free shipping promos
  • volume-based discounts
  • customer-based discounts
  • “stackable” promos & exclusions
  • link-based auto-adds

Checkout

  • express & full checkouts
  • member & non-member ordering
  • fully branded checkout
  • custom SSL certificate
  • pre-checkout recipient mapping
  • pre-checkout shipping estimator
  • Amazon / Google / Paypal checkout

Shipping & Tax

  • multiple-ship to support
  • shipping tables, groups and shippers
  • product-specific shipping rates
  • perishable shipping
  • Fed-Ex / UPS / USPO integration
  • custom shipping API
  • custom tax tables
  • multi-state tax exemptions
  • product-specific tax exemptions
  • Avalara / Exactor / CCH tax integration
  • custom tax API

Payments

  • purchase orders & invoice me
  • gift certificate & card on file
  • electronic check / eCheck.net
  • private label credit cards
  • Authorize.net / Chase Paymentech
  • Paypal’s Payflow Pro & Payflow Link
  • Paypal’s Web Site Payments
  • TransFirst ePay / USA ePay
  • Sterling / Litle & Co / Virtual Merchant
  • Mercury Payment Systems

Order Management

  • order batch processing & PDF copies
  • Mail Order Manager integration
  • Quickbooks Enterprise integration
  • StoneEdge Order Manager integration
  • Natural Order integration
  • NewHaven CMS & Wizard integration
  • OrderMotion integration
  • CoLinear Response integration

Reporting & Analytics

  • Google Analytics integration
  • full traffic & revenue reporting
  • virtual square-inch analysis
  • competitor tracking
  • percentile key metrics ratings

API / External Access

  • secure SOAP-based XML API
  • two-way information exchange
  • update product details
  • update inventory counts
  • batch process orders
  • sync customer data
  • push shipping status/tracking number
